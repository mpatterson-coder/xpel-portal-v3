# XPEL Dealership Portal — V3

Standalone pilot. Runs independently of all XPEL-owned systems.
Everything — users, dealer groups, rooftops, offerings, pricing — is managed
inside the app by admins after a one-time setup.

- `database/schema.sql` + `database/seed.sql` — one-time Supabase setup
- Everything else — the app itself (lives at the repo root on purpose)
- `SETUP.md` — the full path from zero to a live pilot URL
